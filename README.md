PROJECT TOKO DONAT (CRUD LARAVEL)

Deskripsi: Project ini merupakan aplikasi web sederhana berbasis Laravel
yang digunakan untuk mengelola data produk donat. Sistem ini menerapkan
konsep CRUD (Create, Read, Update, Delete) sehingga admin dapat
menambahkan, melihat, mengedit, dan menghapus data donat dengan mudah.
Data disimpan dalam database MySQL dan ditampilkan melalui antarmuka web
yang sederhana.

Teknologi yang digunakan
  - Laravel 13
  - php
  - mysql

TUTORIAL SINGKAT

1.  Clone Project git clone https://github.com/username/nama-project.git
    cd nama-project

2.  Install Dependency composer install npm install npm run dev

3.  Konfigurasi Environment cp .env.example .env

Edit file .env: DB_DATABASE=nama_database DB_USERNAME=root DB_PASSWORD=

4.  Generate Key php artisan key:generate

5.  Migrasi Database php artisan migrate

6.  Jalankan Server php artisan serve

Akses di browser: http://127.0.0.1:8000

CARA MENGGUNAKAN 

-   Tambah Donat: klik tombol tambah, isi data, lalu simpan
-   Lihat Data: data donat tampil di halaman utama
-   Edit Donat: klik edit pada data yang ingin diubah
-   Hapus Donat: klik delete untuk menghapus data
